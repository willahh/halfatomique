Commencé a compiler ces Maps.




Pour commencé on va aprendre a configurer worldcraft.


Cliquez sur l'option "tools" et aller dans "option" puis aller dans "game configurations"
régler les options comme sur le screenshot:                              

Maintenant que vous avez finit de marqué les chemins d'accès aller dans "build programs"
puis configuré les options comme le montre ce screen:                              



     Remarque:  il existe d'autres compilateurs faire cela mais c'est plus complexe.

Maintenant il est temps de compiler la carte.


Pour cela il faut aller dans file et cliqué sur "Run"(F+9).On vous demande de faire une sauvegarde
de votre carte si vous ne l'avez pas fait.Une fois la sauvegarde completé, Une nouvelle fenetre
va s'affichée:
                              
Il faut parametrer comme sur le screen si vous avez fait un effet de lumiere(voir tut gérer les effets de lumières)
si vous n'en avez pas fait il faut simplement cocher "no" dans l'option "RAD".
                              

Voila pour la compilation normalement cela devrait ce passer sans probleme.

trouver maintenant le fichier "votre_save.bsp" et mettez le dans le répertoir:C:/Sierra/Half-Life/valve/maps.
Lancé Half-Life avec la console activé, cliquez sur console et tapé "votre_save.map"(ne mettez pas les
guillimets).

Si la console ne s'affiche pas dans le menu faite l'opération suivante:
créer un racourci de l'icone half-life, mettez le sur le bureau, allez dans ces propriétés puis
dans racourci et taper la ligne suivante après hl.exe: "-dev -console" en mettant un espace entre
hl.exe et -dev(ne taper pas non plus les guillimets).

Voila, normalement vous êtes dans votre première carte.
